# Code Of Conduct

Be respectful, helpful, and patient.

This is a learning-focused project. Questions, corrections, and suggestions are welcome. Personal attacks, harassment, spam, or intentionally harmful contributions are not.

## Expected Behavior

- explain feedback clearly
- assume good intent
- keep discussions about the work
- help beginners feel allowed to learn

## Unacceptable Behavior

- harassment or insults
- sharing private information
- posting malicious code
- derailing issues or pull requests
