# Project Overview

This repo combines two related computer vision workflows:

- **Object detection**: YOLOv11 notebooks for finding objects in images.
- **Object tracking**: OpenCV tracker scripts for following selected objects across video frames.

## Why Combine Them?

Detection and tracking are often used together. Detection can identify the object, while tracking can follow it frame by frame without running a detector every time.

## What Is Not Included

Large model weights and datasets are intentionally not committed. If a notebook needs them, download them from the original source or generate them locally.
